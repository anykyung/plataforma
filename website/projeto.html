O que é a PAP?

A Prova de Aptidão Profissional (PAP) é o projeto final dos cursos profissionais em Portugal. Consiste na apresentação e defesa de um projeto prático que demonstre as competências adquiridas ao longo do curso, perante um júri de avaliação.

É o culminar de três anos de formação e representa a transição do ambiente escolar para o mundo profissional. A PAP não é apenas uma avaliação — é a oportunidade de mostrar o que se aprendeu e de criar algo com valor real.

O Projeto Original

Inicialmente, a ideia para a PAP era desenvolver um assistente de voz inteligente que pudesse executar comandos através de reconhecimento de voz, utilizando a API do Gemini (inteligência artificial da Google) para processamento de linguagem natural.

O conceito era ambicioso e inovador: um programa que respondesse a comandos falados, capaz de interpretar instruções e executar tarefas automaticamente, tudo através da voz.

Projeto Original (Descartado)

Nome: Assistente de Voz com IA

Tecnologia principal: API Gemini (Google)

Motivo do abandono: A API do Gemini necessária para a funcionalidade de reconhecimento e síntese de voz era paga, tornando o projeto financeiramente inviável para o contexto académico.

Projeto Atual (AccuSoft)

Nome: AccuSoft

Tipo: ERP de Gestão de Frotas

Motivo da escolha: Projeto mais abrangente, com aplicação prática real, utilizando tecnologias gratuitas e conhecimentos consolidados ao longo do curso.

A Decisão de Mudar

Durante o estágio curricular na Domoweb, foi realizada uma reunião para analisar a viabilidade do projeto original. Foi nesse momento que se identificou o principal obstáculo: o custo da API do Gemini.

Para que o assistente de voz funcionasse corretamente, seria necessário subscrever um plano pago da API, o que representava um custo que a estudante teria de suportar. Este fator, aliado à complexidade adicional de integrar um sistema de voz em tempo real, levou à conclusão de que o projeto original não era viável.

Momento chave: Foi durante uma reunião na Domoweb, onde se pesaram os prós e contras, que se decidiu mudar o rumo da PAP para uma plataforma de gestão e logística de frotas — um ERP completo e funcional.

A empresa Domoweb, juntamente com a estudante, propôs então um novo projeto: uma plataforma de gestão e logística de frotas (ERP). Esta solução permitia aplicar um conjunto mais vasto de tecnologias aprendidas ao longo do curso e tinha uma aplicação prática evidente no mercado.

O Novo Projeto: AccuSoft

A AccuSoft nasceu assim da necessidade de encontrar uma alternativa viável, mas rapidamente se revelou uma escolha acertada. O ERP de gestão de frotas permite:

  • Gerir manutenções de forma preventiva, evitando avarias inesperadas.
  • Otimizar rotas para reduzir custos de combustível e melhorar prazos.
  • Auditar frotas garantindo conformidade com normas e regulamentos.
  • Centralizar documentos de viaturas e condutores num único local seguro.

Tecnologias Utilizadas

Para o desenvolvimento da AccuSoft foram utilizadas as seguintes tecnologias:

  • Angular — framework apresentada durante o estágio na Domoweb.
  • Node.js — para o back-end e gestão de rotas da API.
  • C# — linguagem aprendida no 1.º ano do curso, usada na criação da API.
  • PostgreSQL — base de dados relacional para armazenamento seguro dos dados.
  • Visual Studio Code — editor de código utilizado em todo o desenvolvimento.

Para mais detalhes, consulte a página Tecnologias Utilizadas.

Resultado Final

O resultado é uma plataforma completa, funcional e com aplicação prática real. A AccuSoft demonstra as competências adquiridas ao longo do curso profissional e representa o culminar bem-sucedido de três anos de aprendizagem intensa em tecnologia.

A mudança de projeto, inicialmente vista como um obstáculo, revelou-se uma oportunidade para criar algo ainda mais ambicioso e relevante, provando que a adaptabilidade e a capacidade de resolver problemas são competências tão importantes quanto as competências técnicas.
