Ao utilizar a plataforma AccuSoft, está a concordar com as práticas descritas nesta política. Recomendamos a leitura atenta deste documento antes de fornecer quaisquer dados pessoais.

3. Dados Recolhidos

A AccuSoft pode recolher as seguintes categorias de dados pessoais:

  • Dados de identificação: nome completo, endereço de e-mail, número de telefone.
  • Dados profissionais: nome da empresa, cargo, dimensão da frota.
  • Dados de navegação: endereço IP, tipo de dispositivo, browser, páginas visitadas, tempo de sessão.
  • Dados de faturação: informações necessárias para processamento de pagamentos (processadas por terceiros certificados).
  • Dados de localização: dados de geolocalização dos veículos monitorizados pela plataforma.

4. Finalidade do Tratamento

Os dados pessoais recolhidos são utilizados para as seguintes finalidades:

  • Prestação de serviços: fornecer as funcionalidades da plataforma, incluindo rastreamento, relatórios e gestão documental.
  • Comunicação: enviar notificações relacionadas com o serviço, atualizações e suporte ao cliente.
  • Faturação: processar pagamentos e gerir subscrições.
  • Melhorias: analisar a utilização da plataforma para melhorar a experiência do utilizador.
  • Obrigações legais: cumprir com obrigações fiscais e regulamentares aplicáveis.

5. Base Legal para o Tratamento

O tratamento de dados pessoais é realizado com base nos seguintes fundamentos legais, em conformidade com o Regulamento Geral de Proteção de Dados (RGPD):

  • Execução de contrato: para fornecer os serviços contratados.
  • Consentimento: quando o utilizador consente explicitamente, por exemplo, para cookies não essenciais.
  • Interesse legítimo: para melhorar a segurança e o desempenho da plataforma.
  • Obrigação legal: para cumprimento de requisitos fiscais e legais.

6. Partilha de Dados

A AccuSoft não vende nem aluga dados pessoais a terceiros. Os dados podem ser partilhados com:

  • Fornecedores de serviços: processadores de pagamento, serviços de alojamento na nuvem.
  • Autoridades competentes: quando exigido por lei ou para proteger os nossos direitos legais.

7. Armazenamento e Segurança

Os dados pessoais são armazenados em servidores seguros localizados na União Europeia. Implementamos medidas técnicas e organizacionais adequadas para proteger os dados contra acesso não autorizado, alteração, divulgação ou destruição, incluindo encriptação SSL, firewalls e controlos de acesso restrito.

8. Período de Conservação

Os dados pessoais são conservados apenas durante o período necessário para as finalidades para as quais foram recolhidos, ou conforme exigido por lei. Os dados de faturação são conservados pelo período legal de 10 anos. Os dados de utilização são anonimizados após 24 meses de inatividade da conta.

9. Direitos dos Titulares dos Dados

Nos termos do RGPD, o utilizador tem os seguintes direitos:

  • Direito de acesso: solicitar uma cópia dos seus dados pessoais.
  • Direito de retificação: corrigir dados incompletos ou inexatos.
  • Direito de apagamento: solicitar a eliminação dos seus dados («direito a ser esquecido»).
  • Direito de limitação: restringir o tratamento em determinadas circunstâncias.
  • Direito de portabilidade: receber os dados num formato estruturado.
  • Direito de oposição: opor-se ao tratamento para fins de marketing direto.

Para exercer qualquer um destes direitos, contacte-nos através do e-mail anygit25@gmail.com.
