Tecnologias Utilizadas

Esta plataforma foi desenvolvida no âmbito da Prova de Aptidão Profissional (PAP), utilizando um conjunto de ferramentas e tecnologias modernas adquiridas ao longo do curso e durante o estágio curricular na empresa Domoweb.

Editor de Código

Visual Studio Code

Editor de código-fonte leve e poderoso, utilizado diariamente para escrever, depurar e organizar todo o código do projeto. A sua extensa biblioteca de extensões permitiu personalizar o ambiente de desenvolvimento para HTML, CSS, JavaScript, TypeScript e C#.

Utilizado durante todo o curso e no desenvolvimento da PAP
Framework Front-end

Angular

Framework robusto para a construção de aplicações web dinâmicas e reativas. A arquitetura baseada em componentes, a vinculação bidirecional de dados e o sistema de injeção de dependências foram essenciais para criar uma interface de utilizador moderna e responsiva.

Framework apresentada e aprofundada durante o estágio na Domoweb
Ambiente de Execução

Node.js

Ambiente de execução JavaScript do lado do servidor que permite construir aplicações escaláveis e de alto desempenho. Foi utilizado para criar o servidor da aplicação, gerir autenticação, rotas da API e a comunicação com a base de dados.

Aprendido e aplicado durante o desenvolvimento da PAP
Base de Dados

PostgreSQL

Sistema de gestão de base de dados relacional avançado, open source e com mais de 35 anos de desenvolvimento ativo. Foi escolhido para armazenar todos os dados da plataforma — veículos, rotas, utilizadores e documentos — com total segurança e integridade referencial.

Utilizado durante o curso e na PAP para modelação e consultas
Linguagem Back-end

C#

Linguagem de programação orientada a objetos, fortemente tipada e desenvolvida pela Microsoft. Foi a primeira linguagem aprendida no curso e serviu de base para compreender conceitos fundamentais como classes, herança, polimorfismo e encapsulamento.

Aprendida no 1.º ano do curso de Técnico de Gestão de Sistemas Informáticos
Arquitetura

API RESTful

A comunicação entre o front-end (Angular) e o back-end (Node.js + C#) é feita através de uma API RESTful, utilizando o formato JSON para troca de dados. Esta arquitetura garante que a plataforma seja modular, escalável e de fácil manutenção.

Implementada para integrar todas as camadas da aplicação
